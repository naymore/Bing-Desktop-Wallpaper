# Bing-Desktop-Wallpaper
Sets your Windows desktop background image to the latest Bing wallpaper (see http://www.bing.com/)

**Options:**
- Specify region (default: Current ui culture)
- Specify image resolution (default: 1920x1080): You can also set the resolution to 1920x1200
- Specify image orientation (default: Fill): Fill, Stretch, Center, ...

**Note:**
*The wallpapers are locally stored in a "Bing Wallpapers" folder within your profile's "My Pictures" folder.*
